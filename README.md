# DRL-SDN Load Balancer — LaTeX Presentation

This repository contains the LaTeX Beamer presentation for the project:

Deep Reinforcement Learning-Based Load Balancer in Software-Defined Networking (SDN)

---

## Requirements (Ubuntu)

Install LaTeX distribution:

sudo apt update  
sudo apt install texlive-full

---

## Compile the Presentation
 
1. Install extension: LaTeX Workshop  
2. Open this folder in VS Code  
3. Open main.tex  
4. Press Ctrl + Alt + B to build  
5. View the generated PDF inside VS Code
